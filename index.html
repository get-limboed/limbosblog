My First Post - Full Content

Welcome to my blog! I'm so excited to start sharing my thoughts and experiences with you all.

As I mentioned in the preview, I'm passionate about retro computing. My journey began when I found an old Windows 98 machine at a garage sale. There was something magical about booting it up and hearing that familiar startup sound.

Since then, I've collected various systems including:

  • Windows 95/98 machines
  • Classic Macintosh systems
  • Commodore 64
  • Various DOS-based PCs

In future posts, I'll be diving deeper into each of these systems, sharing setup guides, software recommendations, and my personal experiences with them.

I'm also planning to document my attempts at getting modern web browsers running on these old systems - which as you can imagine is quite a challenge!

Stay tuned for more content, and feel free to explore the Minesweeper game in the Start Menu - a classic Windows 98 game that I've integrated into this blog for your enjoyment!

My Retro Computing Collection

Here's a more detailed look at my retro computing collection:

1. Windows 98 SE Machine

My pride and joy - a Dell Dimension XPS T500 running Windows 98 SE. Specs:

  • 500MHz Pentium III
  • 256MB RAM
  • 20GB HDD
  • NVIDIA RIVA TNT2 (32MB)
  • Sound Blaster Live! sound card

This machine runs most late-90s games perfectly and is great for nostalgia trips.

2. iMac G3 (Tray Loading)

The iconic Bondi Blue iMac that saved Apple. This one runs Mac OS 9.2.2 and is perfect for classic Mac gaming.

3. IBM PS/2 Model 30

A classic DOS machine with an 8086 processor and MCGA graphics. Great for early DOS games and seeing how computing was in the late 80s.

4. Commodore 64

Complete with original monitor and 1541 floppy drive. Nothing beats the sound of that floppy drive loading!

I'm always on the lookout for new additions to my collection. If you have any recommendations for what I should look for next, let me know!
